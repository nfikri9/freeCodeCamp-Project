    • INTRODUCTION

    JavaScript is the programming language of the Web.
    JavaScript is used to program the behavior of web pages.
    The purpose of JavaScript:

    • Can changes HTML contents
    • Can changes HTML attribute values
    • Can changes HTML styles (CSS)
    • Can modifies the visibility of HTML elements

    Installation

    Developer don't have to get or download JavaScript.
    JavaScript is already running in browser on computers, tablets, and smart-phones.
    JavaScript is free to use for everyone.
    There are multiple ways to include JavaScript:

    • Inside <script> element
      • Scripts can be placed in the <head>, or in the <body> element of an HTML page, or in both.
      • Since script interpretation slows down the display, it is recommended placing scripts at the bottom of the <body> element to improve the display speed.
      • Example: 
         <head> 
           <script> // My JavaScript codes </script> 
         </head> 
         <body> 
           <!--My HTML contents--> 
           <script> // My JavaScript codes </script> 
         </body>
    • External JavaScript files
      • External scripts are practical when the same code is used in many different web pages.
      • JavaScript files have the file extension like myfilename.js.
      • To use external file, use <script src="myfilename.js"></script> (use multiple <script> elements for multiple JavaScript files)
      • Placing scripts in external files has some advantages:
        • It separates HTML and JavaScript
        • It makes HTML and JavaScript easier to read and maintain
        • Cached JavaScript files can speed up page loads

    SYNTAX

    Syntax refers to the rules that define the structure of any language.
    Every statement is ended with semicolon.
    There are many type of statements in JavaScript:
    var x; // Example of declaration statement
    var x = 5; // Example of initialization statement
    x = 5 + 4; // Example of expression statement
    if(x=6){} // Example of conditional/logical statement

    Identifiers

    It is a name for an identifier.
    The following are the rules in naming identifiers:

    • MyIdentifier // Applies PascalCase, where every first letter of every word is uppercase
    • myIdentifier // Applies camelCase, where every first letter of the first word is lowercase and the first letter of the next word is uppercase
    • caseSensitive // Applies case sensitive (MyIdentifier is not the same identifier as myIdentifier)
    • $myIdentifier_1 // Initial letter can be any character, underscores, and dollar sign (subsequent characters can contain letters, digits, underscores, and dollar signs)
    • myIdentifier-1 // ERROR (Can't contain any hyphens as the statement view that as an subtration operator or negative values)
    • 4myIdentifier // ERROR (Can't start with digits)
    • var // ERROR (Can't use reserved keyword as an identifier)

    Values

    The JavaScript syntax defines two types of values:

    • Literals
      • There are fixed values, which can be written in any data type (100 or 100.0 as in number, "John" as in string, true as in boolean, etc.)
      • Example: total = 5 + 6; // 5 and 6 are literals (written directly as value)

    • Variables
      • There are variable values, which each variables stored their own values.
      • Example: total = x + y; // x and y are variables (written as variables, to represent their values respectively)

    Comment

    It can be used to explain JavaScript code, make it more readable, or prevent execution when testing alternative code.
    There are 2 types:

    1. Single Line Comments, // This is my single line comment (the comment start after double dashed until the end of a line)
    2. Multi-line Comments, /* This is my multi-line comment */ (the comment start with /* and end with */)

    OPERATOR

    An expression statement is consist of variables, values and operators, which computes to a value (called computation/evaluation)
    myVariable = myValue1 + myValue2; // myValue1 and myValue2 are values that performed addition operators, + and then assigned to variable myVariable with operator simple assignment, = as an evaluation

    Arithmetic Operators

    It is used to perform arithmetic on numbers (can be written with literals or variables).
    The precedence is from left to right applying BODMAS (bracket, power off, division/multiplication, addition/subtraction).
    Concatenation is addition of values for string (works for addition only).

    let x = 5; let y = "Hello";
    OPERATORS DESCRIPTIONS EXPRESSIONS RESULTS
    Addition Performs addition operation x + 1;
    y + "World";    		 // x = 6
    // y = HelloWorld
    Pre-increment Performs addition operation ++x; // x = 6 (x adds 1 on the same line)
    Post-increment Perform addition operation x++; // x = 5 (x adds 1 on the next line)
    Subtraction Perform subtraction operation x - 1; // x = 4
    Pre-decrement Perform subtraction operation --x; // x = 4 (x subtracts 1 on the same line)
    Post-decrement Perform subtraction operation x--; // x = 5 (x subtracts 1 on the next line)
    Multiplication Perform multiplication operation x * 3; // x = 15
    Exponentiation Perform multiplication operation x ** 3; // x = 125 (multiplies itself 3 times)
    Division Perform division operation x / 3; // x = 1.6666666666666667 (takes quotient as evaluation)
    Modulo Perform division operation x % 3; // x = 2 (takes remainder as evaluation)

    Comparison Operators

    It is used in comparison statements to determine equality or difference between variables or values.
    Commonly used to test for true or false.

    let x = 5;
    OPERATORS DESCRIPTIONS COMPARISON STATEMENTS RESULTS
    Equal to Comparing the equality of data by values x == 8;
    x == 5;
    x == "5";    		 // false (x is not equal to 8 in value)
    // true (x is equal to 5 in value)
    // true (x is equal to 5 in value, data type neglected)
    Not equal to Comparing the equality of data by values x != 8;
    x != 5;
    x != "5";    		 // true (x is not equal to 8 in value)
    // false (x is equal to 5 in value)
    // false (x is equal to 5 in value, data type negelected)
    Strict equal to Comparing the equality of data by both values and data types x === 8;
    x === 5;
    x === "5";    		 // false (x is not equal to 8 in value, equal in data type)
    // true (x is equal to 5 in both value and data type)
    // false (x is equal to 5 in value, not equal in data type)
    Strict not equal to Comparing the equality of data by both values and data types x !== 8;
    x !== 5;
    x !== "5";    		 // true (x is not equal to 8 in value, equal in data type)
    // false (x is equal to 5 in both value and data type)
    // true (x is equal to 5 in value, not equal in data type)
    Greater than Comparing the difference of data by values x > 8;
    x > 3;
    x > 5;    		 // false (x is not greater than 8)
    // true (x is greater than 3)
    // false (x is not greater than 5)
    Less than Comparing the difference of data by values x < 8;
    x < 3;
    x < 5;    		 // true (x is less than 8)
    // false (x is not less than 3)
    // false (x is not less than 5)
    Greater than or equal to Comparing the difference or equality of data by values x >= 8;
    x >= 3;
    x >= 5;    		 // false (x is not greater than or equal to 8)
    // true (x is greater than or equal to 3)
    // true (x is greater than or equal to 5)
    Less than or equal to Comparing the difference or equality of data by values x <= 8;
    x <= 3;
    x <= 5;    		 // true (x is less than or equal to 8)
    // false (x is not less than or equal to 3)
    // true (x is less than or equal to 5)

    Logical Operators

    It is used to determine the logic between variables or values.
    Commonly used to test for true or false.

    let x = 6; let y = 3;
    OPERATORS DESCRIPTIONS LOGICAL STATEMENTS RESULTS
    Logical AND Return true if all logics are true (x < 10) && (y > 1);
    (x > 10) && (y > 1);    		 // true (all logics are true)
    // false (one of the logic is false)
    Logical OR Return true if atleast one of logics is true (x == 5) || (y == 7);
    (x == 6) || (y == 7);    		 // false (all logics are false)
    // true (one of the logic is true)
    Logical NOT Invert the result of the logic !(x == 10);
    !(x == 6);    		 // true (the logic is false but then inverted)
    // false (the logic is true but then inverted)

    Bitwise Operators

    JavaScript stores numbers as 64 bits floating point numbers, but all bitwise operations are performed on 32 bits binary numbers.
    Before a bitwise operation is performed, JavaScript converts numbers to 32 bits signed integers.
    After the bitwise operation is performed, the result is converted back to 64 bits JavaScript numbers.

    let x = 5;
    OPERATORS DESCRIPTIONS OPERATION BINARY OPERATION RESULTS
    Bitwise AND Sets each bit to 1 if both bits are 1 x & 1; 0101 & 0001; // 0001 → 1
    Bitwise OR Sets each bit to 1 if one of the bits is 1 x | 1; 0101 | 0001; // 0101 → 5
    Bitwise XOR Sets each bit to 1 if only one of the bits is 1 x ^ 1; 0101 ^ 0001; // 0100 → 4
    Bitwise NOT Inverts all the bits ~ x; ~ 0101; // 1010 → 10
    Bitwise Left Shift Shifts left by pushing zeros in from the right and let the leftmost bits fall off x << 1; 0101 << 1; // 1010 → 10
    Bitwise Right Shift Shifts right by pushing copies of the leftmost bit in from the left, and let the rightmost bits fall off x >> 1; 0101 >> 1; // 0010 → 2
    Bitwise Unsigned Right Shift Shifts right by pushing zeros in from the left, and let the rightmost bits fall off x >>> 1; 0101 >>> 1; // 0010 → 2

    Assignment Operators

    It is used to assign values to variable.
    Concatenation also works for assignment operators (works for simple and addition assignment only).

    let x = 5; let y = "Hello";
    OPERATORS EXPRESSIONS IDENTICAL EXPRESSIONS RESULTS
    Simple Assignment x = 5;
    y = "Hello";    		 // x = 5
    // y = Hello
    Addition Assignment x += 1;
    y += "World";    		 x = x + 1;
    y = y + "World";    		 // x = 6
    // y = HelloWorld
    Subtraction Assignment x -= 1; x = x - 1; // x = 4
    Multiplication Assignment x *= 3; x = x * 3; // x = 15
    Exponentiation Assignment x **= 3; x = x ** 3; // x = 125
    Division Assignment x /= 3; x = x / 3; // x = 1.6666666666666667
    Modulo Assignment x %= 3; x = x % 3; // x = 2
    Bitwise AND Assignment x &= 1; x = x & 1; // x = 1
    Bitwise OR Assignment x |= 1; x = x | 1; // x = 5
    Bitwise XOR Assignment x ^= 1; x = x ^ 1; // x = 4
    Left Shift Assignment x <<= 1; x = x << 1; // x = 10
    Right Shift Assignment x >>= 1; x = x >> 1; // x = 2
    Unsigned Right Shift Assignment x >>>= 1; x = x >>> 1; // x = 2
    Logical AND Assignment x &&= y; x = x && (x=y);
    Logical OR Assignment x ||= y; x = x || (x=y);
    Nullish Coalescing Assignment x ??= 1; x = x ?? (x=1); // x = 5 (x = 1 if x undefined)

    STATEMENT

    In a statement, keyword plays biggest role in JavaScript.
    Keyword is used to determines type of identifiers, and also determine actions to be performed in a statement.

    • Variable = It is a container for storing data.
    • Function = ???

    Declaration / Initialization statement

    • Keyword var determines the identifier as a variable.
      var a; // a=undefined (Identifier a is declared by keyword var without assigned value)
      var x = 5; // x=5 (Identifier x is initialized by keyword var with assigned value)
      var x = 6; // x=6 (Syntax allowing identifier x to be re-declared within same scope)
      x = 6 + 1; // x=7 (Syntax allowing identifier x to change value)
      {var x = 8;} // x=8 (Syntax allowing identifier x to be re-declared within different block scope)
      console.log(x); // x=8 (Keyword var is global scope, the value can be accessed between the scope)

    • Keyword let determines the identifier as a variable.
      let a; // a=undefined (Identifier a is declared by keyword let without assigned value)
      let x = 5; // x=5 (Identifier x is initialized by keyword let with assigned value)
      let x = 6; // error (Syntax not allowed identifier x to be re-declared within same scope)
      x = 6 + 1; // x=7 (Syntax allowing identifier x to change value)
      {let x = 8;} // x=8 (Syntax allowing identifier x to be re-declared within different block scope)
      console.log(x); // x=7 (Keyword let is block scope, the value can't be accessed between the scope)

    • Keyword const determines the identifier as a variable.
      const a; // error (Syntax not allowed identifier a declared by keyword const without assigned value)
      const x = 5; // x=5 (Identifier x is initialized by keyword const with assigned value)
      const x = 6; // error (Syntax not allowed identifier x to be re-declared within same scope)
      x = 6 + 1; // error (Syntax not allowed identifier x to change value)
      {const x = 8;} // x=8 (Syntax allowing identifier x to be re-declared within different block scope)
      console.log(x); // x=5 (Keyword const is block scope, the value can't be accessed between the scope)

    • Keyword function determines the identifier as a function.
