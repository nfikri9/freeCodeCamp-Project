    - Element <video> = Specifies video content.
    - Element <source> = Specify alternative video files which the browser may choose from. (The browser will use the first recognized format)
    - Element <track> = Specifies text tracks for video.
    - mytext will only be displayed in browsers that do not support the <video> element.
    - Attribute height="myvalue" = Specifies the height of a video player.
    - Attribute poster="myvalue" = Specifies an image to be shown while the video is downloading, or until the user hits the play button.
    - Attribute preload="myvalue" = KIV
    - Attribute src="myvalue" = Specifies the URL of the video.
    - Attribute width="myvalue" = Specifies the width of a video player.
    - Attribute autoplay = The video will automatically start playing as soon as webpage is loaded.
    - Attribute controls = Specifies controls that displayed features of play, pause, volume, seeking, fullscreen toggle, captions/subtitles, track.
    - Attribute loop = Specifies video will start over again, every time it is finished.
    - Attribute muted = Specifies video should be muted.
    - Attribute type="myvalue" = Specifies the video type. (myvideo/mp4, myvideo/ogg, myvideo/webm)
    - Attribute default = Specifies that the track is to be enabled if the user's preferences do not indicate that another track would be more appropriate.
    - Attribute kind="myvalue" = Specifies the kind of text track. (captions, chapters, descriptions, metadata, subtitles)
    - Attribute label="myvalue" = Specifies the title of the text track.
    - Attribute srclang="myvalue" = Specifies the language of the track text data. (required if kind="subtitles")
    - Element <video>, <source>, <track> support Global Attributes, Event Attributes, HTML DOM.
